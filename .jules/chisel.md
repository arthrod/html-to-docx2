## 2024-05-30 - [toHTML/toTestFormat options] **Learning:** The options for functions like `toHTML` and `toTestFormat` in `src/vendor/justjshtml/serialize.ts` were untyped destructured assignments with default values causing `TS7022` and `TS7023` implicit any errors and missing property errors hidden by `@ts-expect-error`. Additionally, the constant `FOREIGN_ATTRIBUTE_ADJUSTMENTS` structure explicitly revealed the typing for `foreignAttributeAdjustments`. **Action:** Extracted the implicit inline `{}` param options into exported `ToHTMLOptions` and `TestFormatOptions` interfaces with explicitly typed optional properties using `Record<string, [string | null, string, string]>` for complex constants.
